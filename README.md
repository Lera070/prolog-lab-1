**Запросы и результаты:**

1. Определить всех сыновей матери Алены
   - Запрос: `сын(Кто, алена).`
   - Результат: `Кто = сергей`

2. Определить всех детей пары родителей (Иван и Алена)
   - Запрос: `дети(иван, алена, Ребенок).`
   - Результаты: `Ребенок = сергей`, `Ребенок = лера`

3. Определить родителей Игоря
   - Запрос: `родитель(Родитель, игорь).`
   - Результаты: `Родитель = сергей`, `Родитель = людмила`

4. Является ли Ксения женщиной?

## Задание 2: Мэри, Бет и фрукты

**Запросы и результаты:**

1. Что любит Бет?
   - Запрос: `любит(бет, X).`
   - Результаты: `X = яблоки`, `X = кукуруза`

2. Любит ли Мэри кукурузу?
   - Запрос: `любит(мэри, кукуруза).`
   - Результат: `true`

3. Какие фрукты известны?
   - Запрос: `фрукт(F).`
   - Результаты: `F = персики`, `F = яблоки`, `F = апельсины`

4. Какого цвета фрукты, которые любят Бет и Мэри?
   - Запрос: `любит(мэри, Фрукт), любит(бет, Фрукт), фрукт(Фрукт), цвет(Фрукт, Цвет).`
   - Результат: `Фрукт = яблоки, Цвет = красный`
